# 🛡️ Smart Cybersecurity Assistant

AI-powered cybersecurity assistant built with **Streamlit** and **Google Gemini API**.

## 🚀 Features

- 📧 **Phishing Email Detection**
  - Analyze suspicious emails
  - Get risk level and explanation

- 💻 **Code Security Analyzer**
  - Detect vulnerabilities in code
  - Get risk level and fixes

- 🔑 **Password Strength Checker**
  - Evaluate password strength
  - Suggestions for improvement

---

## 🧠 Powered By

- Google Gemini API
- Streamlit (for UI)
